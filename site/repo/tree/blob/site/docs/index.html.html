59 <h2>Implementation status</h2>
60 <div class="copy-block">
61 <p>
62 Gossip Protocol is a reference implementation, not a hosted SaaS control plane. The
63 source of truth for operational behavior is the Rust crate, especially
64 <code>src/cli.rs</code>, <code>src/protocol.rs</code>, <code>src/node.rs</code>,
65 <code>src/compute.rs</code>, <code>src/browser.rs</code>, and <code>src/storage.rs</code>.
66 </p>
67 <p>
68 The documentation starts with the core surfaces that are already implemented: wallet
69 creation, genesis creation, validator nodes, signed transactions, peer gossip, monitor
70 heartbeats, browser journeys, sharded compute, storage contracts, public raw bundles,
71 DNS route leases, settlement adapters, probe agents, and storage hosts.
72 </p>
73 </div>
