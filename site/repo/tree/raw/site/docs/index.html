Implementation status

Gossip Protocol is a reference implementation, not a hosted SaaS control plane. The source of truth for operational behavior is the Rust crate, especially src/cli.rs, src/protocol.rs, src/node.rs, src/compute.rs, src/browser.rs, and src/storage.rs.

The documentation starts with the core surfaces that are already implemented: wallet creation, genesis creation, validator nodes, signed transactions, peer gossip, monitor heartbeats, browser journeys, sharded compute, storage contracts, public raw bundles, DNS route leases, settlement adapters, probe agents, and storage hosts.
