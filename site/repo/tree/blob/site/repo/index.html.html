78 </div>
79 <div class="docs-card">
80 <h3><a href="/repo/tree/blob/src/node.rs.html"><code>src/node.rs</code></a></h3>
81 <p>Validator runtime, mempool, peer sync, gossip endpoints, block proposal, approval flow, persistence, and control API.</p>
82 </div>
83 <div class="docs-card">
84 <h3><a href="/repo/tree/blob/src/cli.rs.html"><code>src/cli.rs</code></a></h3>
85 <p>Command-line interface for wallets, genesis, nodes, transactions, swaps, monitors, browser checks, storage, and agents.</p>
86 </div>
87 <div class="docs-card">
88 <h3><a href="/repo/tree/blob/src/compute.rs.html"><code>src/compute.rs</code></a></h3>
89 <p>Compute job schemas, reducers, WASI settings, sandbox policy, cost model, output validation, and shard receipts.</p>
90 </div>
91 <div class="docs-card">
92 <h3><a href="/repo/tree/blob/src/compute_sandbox.rs.html"><code>src/compute_sandbox.rs</code></a></h3>
93 <p>Isolated compute execution, artifact capture, WASI runtime integration, and resource enforcement.</p>
94 </div>
95 <div class="docs-card">
96 <h3><a href="/repo/tree/blob/src/browser.rs.html"><code>src/browser.rs</code></a></h3>
97 <p>Browser journey package schema, runtime profile validation, artifact policy, package hashing, and runner IO.</p>
98 </div>
99 <div class="docs-card">
100 <h3><a href="/repo/tree/blob/src/storage.rs.html"><code>src/storage.rs</code></a></h3>
101 <p>Storage bundle builder, chunk server, Merkle proofs, encrypted bundles, public raw bundles, and domain gateway serving.</p>
102 </div>
103 <div class="docs-card">
104 <h3><a href="/repo/tree/blob/src/agent.rs.html"><code>src/agent.rs</code></a></h3>
105 <p>Delegated probe, browser, and compute agent runtime with lease verification and signed delegated receipts.</p>
106 </div>
107 <div class="docs-card">
108 <h3><a href="/repo/tree/blob/src/wallet.rs.html"><code>src/wallet.rs</code></a></h3>
109 <p>Wallet files, Ed25519 signing, verification helpers, and typed signature methods for protocol objects.</p>
110 </div>
111 <div class="docs-card">
